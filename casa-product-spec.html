The Opportunity

Why China Needs CASA

The household management app space in China is completely untapped. Heima proved the concept works in the West. No one has built a polished, China-native version — until now.

The Core Insight

Chinese families are structurally different from Western ones — they often have 3 generations under one roof, grandparents actively managing childcare and cooking, and dual-income parents with near-zero spare time. The pain of household coordination is more acute in China, not less. Yet no good app exists. CASA fills that gap.

👴👵
三代同堂 — 3-Generation Households

Over 70% of elderly parents aged 80+ in China live with their children or grandchildren. Grandparents handle cooking, school pickups, and cleaning — but coordination between tech-savvy young adults and elderly relatives breaks down constantly. No existing app handles this age gap.

⚖️
家务不平等 — The Chore Inequality Crisis

Chinese women spend nearly 4 hours daily on housework vs. 1.5 hours for men — one of the most extreme gender gaps in East Asia. Academic research (CFPS longitudinal data) shows having a child dramatically widens this gap further. This is a top-trending social issue on 小红书, 知乎, and 抖音. CASA makes invisible labor visible.

👶
二孩/三孩 Policy — Growing Family Complexity

China's government is actively encouraging families to have 2–3 children through policy incentives (extended maternity leave, childcare subsidies). More children = exponentially more household complexity. A family with 2 children, 2 grandparents, and 2 working parents needs coordination tools, not just group chats.

📱
WeChat-First Infrastructure

China's 1.3 billion WeChat users already coordinate family life on the platform — but it's manual, unstructured, and easily lost in chat history. An app that lives inside WeChat as a Mini Program has zero friction to adoption. Grandparents who already use WeChat need no new app to install or learn.

🏙️
Urban Middle-Class Spending Shift

Chinese urban consumers now regularly pay for premium apps. Keep (fitness), BOSS直聘 (jobs), 小睡眠 (sleep), and 得到 (knowledge) all monetize at ¥100–300/year successfully. The ¥98/year CASA premium tier is well within the "easily justified" spend range for urban families managing a home together.

📉
No Dominant Competitor Exists

Heima (Iceland) proved product-market fit in the West. China has no equivalent: 做家务 app is basic and untouched; 钉钉/飞书 are work tools; WeChat groups are unstructured. This is a rare blue-ocean opportunity in one of the world's largest digital consumer markets — ¥165B home appliance market (Statista 2025) shows strong household investment culture.

Target Users

Who Uses CASA

CASA must work for everyone in a Chinese family — from a 30-year-old working mom on an iPhone to a 68-year-old grandfather on a Huawei with large text enabled. One app, four experiences.

👩
张三, 32
Working mother · Software Engineer
😤 "I end up doing 80% of the housework while also managing shopping, remembering everyone's schedules, and coordinating with my in-laws. My husband has no idea what needs doing."
✅ Needs: Fair task visibility, one-tap assign to husband, weekly family progress summary, AI fairness report to share
👨
李四, 35
张三's husband · Marketing Manager
😓 "I want to help but I never know what needs doing. I get reminded verbally but forget. I'd much rather check a list on my phone — I respond to systems, not nagging."
✅ Needs: Simple daily task list, push notification reminders, points for completed chores, no friction to get started
👴
王五, 67
Retired grandfather · Helps with childcare & cooking
😕 "My daughter-in-law sends me tasks on WeChat but I forget them. Young people's apps are all too small and complicated. I just want to know: what do I need to do today?"
✅ Needs: Giant text, WeChat-based reminders (no new app to install), very simple 1-tap task completion, Elder Mode interface
🧒
马六, 9
Primary school student · Youngest 家庭成员
😶 "I never know which chores are mine this week and mom always forgets to give me my allowance. I wish there was a game to track it."
✅ Needs: Fun points and badge system, allowance tracking tied to chore completion, simple colorful kid-friendly interface

Design Principle: One App, Four Distinct Experiences

The same app must serve a 32-year-old iPhone power user and a 67-year-old Android user who barely uses apps. CASA solves this with role-based UI modes — each 家庭成员 gets an interface tuned to their age, tech fluency, and responsibilities. This is something Heima doesn't do, and it's a key differentiator for China's 三代同堂 market.

Product Features

What CASA Does

Features are tagged by release phase. Build MVP fast, validate with real families, then layer on advanced features based on actual user data.

MVP — Month 1–3 V2 — Month 4–6 V3 — Month 7–12 🇨🇳 China-unique

Core Features

📋
家务看板 Chore Board MVP

Visual weekly chore chart. Assign tasks to any 家庭成员. Daily, weekly, monthly repeat cycles. Color-coded by person. One-tap complete with satisfying animation.

👨‍👩‍👧‍👦
家庭成员 Family Profiles MVP

Add all 家庭成员s with roles (妈妈, 爸爸, 爷爷, 孩子). Each gets their own task view. Invite via WeChat in one tap — no phone number needed.

🏆
积分系统 Points & Rewards MVP

Every completed chore earns points. Family leaderboard. Kids earn points redeemable for allowance. Parents set custom rewards (screen time, outings, toys).

🔔
智能提醒 Smart Reminders MVP

Push notifications + WeChat template message reminders. Gentle nudge before deadline. Grandparents receive via WeChat — no app installation required.

📊
家务报告 Weekly Reports MVP

Beautiful weekly summary: who did what, fairness percentage, total family points. Shareable to WeChat Moments — "我们家本周家务贡献 78%".

🛒
购物清单 Shopping Lists MVP

Shared real-time shopping list. Any 家庭成员 can add items. Grandpa can check items off at the market. Real-time sync via Socket.io.

China-Unique Features

👴
老人模式 Elder Mode 🇨🇳 MVP

One tap switches to giant text, simplified UI with only 3 functions: see my tasks, mark done, call family. High contrast. Designed with grandparents in beta testing. No other household app does this.

🧧
春节特别模式 Spring Festival Mode 🇨🇳 V2

Pre-loaded 大扫除 (Spring Festival cleaning) checklist. Special bonus points during 年前. Festive terracotta & gold UI theme. The most share-worthy feature of the year.

🤖
AI 家务助手 AI Chore Assistant 🇨🇳 V2

Input family size, apartment size, work schedules. AI generates a fair weekly chore plan. Detects imbalance and suggests redistribution. Powered by Claude API with 文心一言 fallback.

🎁
积分兑换 Points → Real Rewards 🇨🇳 V3

Partner with JD.com or Meituan — family points can redeem real vouchers. Massive engagement driver unique to China's super-app ecosystem.

Advanced Features

📦
家庭库存 Home Inventory V2

Track household supplies. Auto-alert when running low. Connect to JD/Taobao to reorder with one tap.

💰
家庭账本 Family Budget V3

Track shared household expenses. Who paid for what. Monthly summaries. Chinese families are highly financially collaborative.

📅
家庭日历 Shared Calendar V2

Kids' school events, doctor appointments, birthdays. Syncs with phone calendar. Built-in 节气 Chinese solar terms events.

UI Concept

App Screen Concepts

Warm, grounded, and purposeful. Full phone mockups with bottom navigation bar — 5 tabs, each with a hand-drawn SVG icon. The 报表 tab includes a top tab bar for multiple report views.

报表 Tab Detail

The 报表 section has 4 tabs accessible from the top bar, each showing a different analytical view of family data.

Tab 1
周报告 Weekly Report

Chore completion counts per person, points earned, fairness percentage donut chart, shareable card for WeChat Moments.

Tab 2
积分榜 Scoreboard

Ranked family leaderboard by points. Weekly and all-time views. Animated rank changes. Badges for streaks and milestones.

Tab 3
公平指数 Fairness

AI-generated fairness score with breakdown by task category (cleaning, cooking, childcare). Tracks improvement or decline week over week.

Tab 4
趋势 Trends

4-week sparkline chart per 家庭成员. Seasonal patterns (Spring Festival spike). Identifies who's improving vs. declining over time.

Engineering

Tech Stack

Chosen for speed of development, China-compatibility, and long-term scalability. Every tool is vetted to work inside China's firewall. Nothing that's blocked or slow in the mainland.

Frontend · Mobile
React Native 0.73+
One codebase → iOS + Android. Large ecosystem, fast iteration. NativeWind for styling.
Frontend · WeChat
Taro.js + 微信小程序
Compiles to WeChat Mini Program + React Native from one codebase. Highest reach, zero install friction in China.
Backend
Node.js 20 + Express
Fast to build, easy to hire for. REST API shared between app and mini program. BullMQ for job queues.
Database
PostgreSQL 15 + Prisma
Reliable relational DB — ideal for family/task/points data. Prisma for type-safe ORM with migration support.
Real-time Sync
Socket.io + Redis 7
Real-time shopping list and chore updates. All 家庭成员s see changes instantly. Redis for session + cache.
Auth
WeChat OAuth + JWT
Login with WeChat — zero friction for Chinese users. No new password to remember. JWT for API sessions.
AI Layer
Claude API + 文心一言
Claude for chore suggestions and fairness analysis. Baidu ERNIE (文心一言) as China-hosted fallback when needed.
Notifications
JPush + APNs + 微信模板
JPush for Android OEMs (Huawei, OPPO, Xiaomi). APNs for iOS. WeChat template messages reach grandparents directly.
Hosting
阿里云 ECS + RDS + OSS
Must host in China for performance + ICP compliance. Alibaba Cloud ECS (compute), RDS (PostgreSQL), OSS (storage).
Analytics
微信数据分析 + Mixpanel
WeChat's built-in analytics for mini program metrics + Mixpanel for custom user funnels and retention tracking.
Payments
微信支付 + 支付宝
Both required. WeChat Pay for Mini Program flow, Alipay for app store. No credit cards needed — both cover 99% of users.
Legal / Compliance
ICP备案 + 数据安全法
ICP license required to operate commercially in China — budget 4–8 weeks. Also comply with PIPL and Data Security Law.

⚠️ Critical China Infrastructure Warning

Never use Google Firebase, Google Analytics, or AWS us-east for China users — these are blocked or extremely slow behind the Great Firewall. Host everything on 阿里云 (Alibaba Cloud) inside China with an ICP license. Start the ICP备案 application on Day 1 of the project — approval takes 4–8 weeks and nothing can go live without it. This is the #1 mistake foreign developers make when building for China.
